  • eLife
    October 24, 2017 by Naomi Penfold (eLife), Adrià Mercader (OKI), and Jo Barratt (OKI)
    pilot

    Investigating the applicability of the goodtables library to facilitate data validation in the field of Life and Biomedical Sciences and make a case for the reusability of data shared with eLife as additional or supporting files.

  • Tool Fund Grantee: Georges Labrèche
    October 24, 2017 by Georges Labreche
    grantee-profiles

    This grantee profile features Georges Labreche for our series of Frictionless Data Tool Fund posts, written to shine a light on Frictionless Data’s Tool Fund grantees, their work and to let our technical community know how they can get involved.

  • Zegami
    September 28, 2017 by Roger Noble and Andrew Stretton
    case-studies

    As a visual data exploration and analytics platform, Zegami makes the exploration of large collections of image rich information quick and simple.

  • Causa Natura - Pescando Datos
    August 15, 2017 by Gabriela Rodriguez (Causa Natura/Engine Room), Adrià Mercader (OKI), Jo Barratt (OKI), Eduardo Rolón (Causa Natura)
    pilot

    Using goodtables to improve quality of data to support fisher communities and advocacy groups.

  • Center for Data Science and Public Policy, Workforce Data Initiative
    August 15, 2017 by Frictionless Data
    case-studies

    Supporting state and local workforce boards in managing and publishing data.

  • University of Cambridge - Retinal Mosaics
    August 15, 2017 by Stephen Eglen (University of Cambridge), Dan Fowler (OKI)
    pilot

    Investigating the applicability of the Data Package concept to facilitate data reuse in the field of Computational Neuroscience.

  • Collections as Data Facets - Carnegie Museum of Art Collection Data
    August 9, 2017 by Dan Fowler
    case-studies

    Use of Frictionless Data specifications in the release of Carnegie Museum of Arts’ Collection Data for public access & creative use

  • Template for Tutorials
    August 9, 2017 by Frictionless Data

    This post provides you with a template for writing Frictionless Data tutorials. Specifically, tutorials of the form: How to do X thing using Y Frictionless Data tool.

  • Pacific Northwest National Laboratory - Active Data Biology
    June 26, 2017 by Sam Payne (PNNL), Joon-Yong Lee (PNNL), Dan Fowler (OKI)
    pilot

    Using goodtables to validate metadata stored as part of an biological application on GitHub.

  • The Data Retriever
    May 24, 2017 by Ethan White
    case-studies

    The Data Retriever is a package manager for data. It downloads, cleans, and stores publicly available data, so that analysts spend less time cleaning and managing data, and more time analyzing it.
